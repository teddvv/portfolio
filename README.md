# Portfolio — Teddy Pategou

Mon portfolio perso, fait dans le cadre du module Web d'Epitech Strasbourg.

Site en ligne : https://teddvv.github.io/portfolio/

## Stack

React + Vite + Tailwind. C'est tout.

## Lancer en local

```bash
npm install
npm run dev
```

Ça ouvre le site sur `http://localhost:5173`.

## Build de prod

```bash
npm run build
```

Les fichiers sortent dans `dist/`.
